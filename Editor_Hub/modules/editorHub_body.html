v-1.1.4

June 2023

changed - Audio Player.

The Audio Player has got a new and improved look, you can now use the space bar to play or pause your current file

The Audio Player's default volume was 100%, which resulted in uncomfortably loud previews and forcing the user to change volume levels while previewing.
The default volume is now set at 30%, which should make it more pleasant to interact with.

changed - Song Manager

The Song Manager tab has been reworked and it should be more comfortable to interact with this tool.

Each item has 2 buttons that can be revealed by hovering on the right edge of the item box. These buttons allow you to preview or import the file.

Song Manager Tutorial

new - SFX Manager

The new SFX Manager tab has been added in the Audio Tools workspace. It works identically to the newly reworked Song Manager tab

SFX Manager Tutorial

new - Place Action

Now, in addition to being able to preview and import, you also have the option to place a file directly on the timeline.

The files are placed on tracks that are created and managed by the Editor Hub, you can customize these track options in the new Settings tab to give different names or set custom key binds for different tracks.

Refer to the tutorial for a more in depth explanation.

new - Audio Tool Settings Tab

The Editor Hub interacts with Adobe Premiere in dynamic and complex ways. This opens up a need for customization.

In the new Settings tab within the Audio Tools workspace you'll be able to customize settings such as the default audio player volume, default track names, where to download your imports custom keybinds to operate your actions and more.

Audio Tool Settings Tutorial

new - Keyboard Shortcuts

While fetching song tracks or sfx audio files, you can now make use of keyboard shortcuts to preview, import, or place your file directly on the timeline

Go to the Settings Tab in the Audio Tools workspace to customize any of these Key Binds

v-1.0.0

Jan 2023

- Editor Hub

The Editor Hub is a Premiere extension developed to help Skill Capped piecers and editors automate repetitive and technically tedious tasks as well as to standardize editing frameworks, workflows and asset libraries.

- Audio Tools Workspace

The Audio Tools workspace is meant to host a variety of audio related tools such as dealing with background music, SFX and even audio processing effects such as EQ, compression, etc.
The only tool available on v-1.0.0 is the 'Song Manager'.

- Song Manager

The song Manager allows users to preview tracks without having to import them into the project, giving them access to the official updated list of tracks.
Once you like a track you can download it to the project in order to add it to the timeline. For more details click on the 'Song Manager' tab to the left while holding down 'alt' to open the tutorial for this tool.
